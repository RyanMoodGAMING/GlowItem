# GlowItem

Give an item an enchantment glow.

## Normal Installation

Download the latest version of the plugin from the [releases tab](https://github.com/RyanMoodGAMING/GlowItem/releases).

## Commands & Permissions

`/glowitem` - `glowitem.glow`  
`/glow` - `glowitem.glow`
